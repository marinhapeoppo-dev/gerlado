Quick Start Guide

How to Use Our APIs
  1. Choose an API endpoint from the categories above
  2. Copy the API URL format
  3. Replace parameters with your actual data
  4. Make a GET request to the API
  5. Handle the JSON response in your application

Example Usage

  • JavaScript
  • Python
  • cURL 
// JavaScript Fetch Example
async function getAIChat() {
    try {
        const response = await fetch('/api/gpt?text=Hello, how are you?');
        const data = await response.json();
        console.log(data);
    } catch (error) {
        console.error('Error:', error);
    }
}
# Python Requests Example
import requests

response = requests.get('https://yourdomain.com/api/gpt?text=Hello')
data = response.json()
print(data)
# cURL Example
curl "https://yourdomain.com/api/gpt?text=Hello"
